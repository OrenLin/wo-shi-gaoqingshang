# 🧠 我是高情商 - H5互动游戏

一款以搞笑社交场景为核心的H5互动小游戏，让你在笑声中学习高情商话术！

## ✨ 功能特色

- 🎭 **3个经典场景**：年夜饭催婚、职场送命题、酒局被迫营业
- 🎯 **4级情商选项**：低/中/高/神 四档预设回复
- 🎤 **语音输入**：支持Web Speech API语音识别
- ✍️ **自定义回复**：用户可自由输入，系统智能评分
- 📊 **评分系统**：每场景独立评分 + 最终综合报告
- 🎨 **卡通风格**：Emoji角色 + 流畅动画 + 渐变配色
- 📱 **移动适配**：完全响应式，微信分享友好

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173/

### 构建生产版本

```bash
npm run build
```

构建产物在 `dist/` 目录

## 🎮 游戏流程

1. **首页** - 查看标题和开始按钮
2. **场景选择** - 从3个场景中选择
3. **游戏页面** - 选择预设选项或自定义回复
4. **结果页** - 查看情商评分和点评
5. **报告页** - 完成所有场景后查看综合报告

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **语音识别**: Web Speech API

## 🎯 评分系统

### 预设选项
每个场景4个选项对应固定分数：
- 💀 低情商：15-25分
- 😅 中情商：50-55分
- 😎 高情商：80-85分
- 👑 情商之神：95-98分

### 自定义输入
基于关键词匹配算法评分：
- **加分项**：礼貌用语、幽默元素、转折技巧、长度适中、夸赞对方
- **减分项**：攻击性词汇、过短回复、纯拒绝无缓冲

### 段位体系
- 👑 **情商之神**：90-100分
- 😎 **情商达人**：70-89分
- 😅 **及格选手**：40-69分
- 💀 **社交杀手**：0-39分
